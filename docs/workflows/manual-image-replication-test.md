# 👶 面向新手的 ComfyUI 手动复刻效果验证教程

你好！欢迎来到 ComfyUI 的世界。ComfyUI 是一个基于“节点连线”的 AI 绘图工具。虽然一开始看着像蜘蛛网一样复杂，但只要搞懂了逻辑，您会发现它极其强大。

本教程完全针对**零基础新手**，我们将一步一步演示如何测试**“爆款图风格复刻 + 产品不变形”**的生成效果。

---

## 第一阶段：初识 ComfyUI 界面

1. **打开界面**：在浏览器地址栏输入 `http://<服务器的IP地址>:8188` 然后回车。
2. **认识画布**：
   - 黑色背景的区域就是“画布”。
   - 画布上的一个个小块叫做**节点 (Node)**。
   - 节点左边的点是**输入端口**，右边的点是**输出端口**。
   - 您可以用鼠标左键长按拖动连线，鼠标滚轮缩放画布，鼠标右键拖动平移画布。
3. **认识控制面板**：
   - 在屏幕右侧有一个浮动菜单，最上面的按钮是 `Queue Prompt`（添加到队列：也就是**开始生成图片**的意思）。
   - 下方有 `Load` (加载保存的工作流) 和 `Save` (保存当前连线状态为 JSON 文件)。
   - 最下方有一个 **ComfyUI Manager** 按钮（这是装插件和模型的管家，非常重要）。

---

## 第二阶段：下载并导入现成的“魔法阵 (工作流)”

要实现“图片复刻”，我们需要用到 `FLUX 大模型 + IPAdapter (抓取风格) + ControlNet (控制形状)`。对于新手，自己连线这 20 多根线太容易出错啦！所以我们要站在巨人的肩膀上：**直接导入现成的工作流**。

由于我们使用的是社区整合包，您可以按以下步骤快速获取别人连好的配置：

1. **获取工作流文件 (JSON)**：
   - 如果您手边没有，建议访问 [OpenArt.ai](https://openart.ai/) 或 [LiblibAI (哩布哩布)](https://www.liblib.art/) 搜索 `FLUX IPAdapter ControlNet workflow`，下载一个 `.json` 文件。
   - （或者，如果是团队内部提供的，直接找同事索要一份叫作 `产品复刻版_flux.json` 的文件）。
2. **拖入画布加载**：
   - 把下载好的 `.json` 格式文件，直接用鼠标**按住不放，拖拽到 ComfyUI 的黑色画布上**然后松手。
   - 画布瞬间会变成排布好的复杂网络，这就是导入成功了！

### ⚠️ 红框警告排错
拖入工作流后，如果有节点变成了**红色框边**，说明您的环境里缺少这个插件或模型。
**如何解决？**
1. 点击右侧菜单的 **Manager** 按钮。
2. 点击 **Install Missing Custom Nodes**（安装缺失的自定义节点）。
3. 弹出的列表就是缺少的插件，全部点击 Install 安装。完成后点击重启。
4. 模型如果是红色的（比如 Load Image 里面的模型名变红），需要下载相应的模型放到对应的文件夹下（系统部署时我们已下载）。

---

## 第三阶段：一步步教你填空（核心参数设置）

现在画布上有很多节点，但您**完全不需要管所有的连线**！只要找到这几个关键的“框框 (节点)”去填空即可：

### 操作 1：上传您的【产品白底图】
在画布中寻找名字叫 **Load Image** 并且它连着名为 `ControlNet` (线稿/深度控制) 的节点：
1. 这个框里面会有一张默认图片。
2. 点击图片下方的 `choose file to upload`。
3. 上传您自己产品的清晰白底图或实拍图。
*(它的作用是：让 AI 知道您的产品长啥样，并锁死它的形状轮廓，绝对不能变形)*。

### 操作 2：上传别人的【爆款原图】
在画布中寻找另一个 **Load Image**，它连着名字带有 `IPAdapter`（风格适配器）的节点：
1. 点击 `choose file to upload`。
2. 上传那张您觉得绝美、光影高级、想要复刻其风格氛围的爆款图。
*(它的作用是：提取该图的色彩、光影、材质，将其“传染”到新生成的图上)*。

### 操作 3：输入提示词 (咒语)
找到有两个大文本框的区域，通常它们连着叫 **CLIP Text Encode** 的节点。
1. **正向提示词框 (Positive Prompt)**：
   删掉里面的英文，输入属于您产品的简单描述（必须用英文）。
   > **新手万能公式**： `A professional product photograph of a [您的产品名称, 比如 white ceramic mug], high quality, highly detailed, photorealistic, 8k.`
2. **反向提示词框 (Negative Prompt)**：
   通常在底下那个框。填入：`ugly, deformed, bad quality, low resolution` 即可。

### 操作 4：调整控制强弱（新手必调！）
我们要靠调数值来控制“抄多少”和“变多少”。这是**最关键**的一步：

1. **控制风格像不像 (IPAdapter Weight)** 🎨
   - 找到带有 `IPAdapter Apply` 名字的节点，里面有个 `Weight` 滑块。
   - **新手建议值： 0.70**
   - *（如果调到 1.0，生成的图就会完全照抄爆款图甚至把爆款图的产品也抄进去了；如果调到 0.3，则生成的图风格很平淡。）*

2. **控制产品变不变形 (ControlNet Strength)** 📏
   - 找到带有 `Apply ControlNet` 名字的节点，里面有个 `Strength` 参数。
   - **新手建议值： 0.85**
   - *（如果调到 1.0，产品的轮廓会被死死钉住，边缘可能会显得像一张纸贴上去的（抠图感）；如果调到 0.5，产品就会变形，比如圆口杯子变成了方口。）*

---

## 第四阶段：见证奇迹的时刻

1. 所有的填空完成啦！将鼠标移到屏幕右侧菜单栏。
2. 深呼吸，点击 **Queue Prompt** 按钮一记。
3. 此时，画布上的节点会按照顺序一个一个亮起绿框。这代表 AI 正在这个节点进行计算。
   - *(注意：由于您是第一次运行大模型 FLUX（高达 20GB+），第一次加载进显存可能会卡在加载模型那个节点 5-10 分钟，请耐心喝杯茶等待。只要后台不报错，绿框还在，就说明在进行。之后的图只要 10 秒钟就能跑出来)*。
4. 等绿框跑到最后一个叫 **Save Image** 或 **Preview Image** 的节点时，一张崭新的图片就生成出来了！

👉 **保存图片**：右键点击那张生成好的漂亮大图，选择 `Save Image`（保存图像），下载您的杰作吧。

---

## 🎯 新手复盘与验收标准

如果图片出来了，请对照以下三点判断这次复刻成功与否：

1. **变了吗？** 检查您产品的形状、Logo 有歪七扭八吗？如果有，去把 **ControlNet Strength** 调高 0.05 继续生成一次。
2. **像吗？** 背景的光影氛围有被模仿过来吗？如果风格没过来，把 **IPAdapter Weight** 调高 0.05 再生成一次。
3. **假吗？** 产品放在那里像不像 PS 贴图？如果没有阴影、悬浮在空中，尝试把正向提示词里加上 `natural shadows, lighting matching the background`，并把 ControlNet 的力量往下降一点点（比如从 0.85 降到 0.80）。

祝你在 ComfyUI 的第一次作图顺利！有问题随时看红框警告，或者把您试出来的**最佳参数值**记在小本本上，这将是我们后续数字员工全自动出图的基础配置！
